AI Inovation Project

This is a team project done in React that uses AI to help students on the ISU canvas page. This a chrome extension that appears in the form of a sidebar that can be collapsed which allows users to interact with an AI chatbot to clear up any questions about general logistics of a course. The sidebar also generates a list of upcoming assignments so it can be an easy reminder to keep students from forgeting to complete an assignment. There are also a few buttons that serve certain functionalities which are currently undecided.

Bingo

This is a project done in Java. This is an object-oriented approach for coding the game Bingo. Constructors were used to create the bingo board itself as well as the squares. There are different categories that can be used for the bingo. These categories are organized in a class hierarchy.
